#  Student Management System

A modern Gen-Z styled Student Management System developed using Flask, SQLite, Bootstrap, and custom CSS. The application provides full CRUD operations, search functionality, roll number validation, and a responsive dashboard UI.

---

##  Live Demo

[https://your-render-link.onrender.com](https://student-management-system-vtat.onrender.com)

---

##  Features

- Add Student  
- View Students  
- Edit Student  
- Delete Student  
- Search Students  
- Roll Number Validation  
- Year Selection  
- Responsive Dashboard  
- Gen-Z Glassmorphism UI  
- SQLite Database Integration
- Full CRUD Operations  
- Render Deployment  
- GitHub Version Control

##  Installation

###  Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/student-management-system.git
```

---

###  Open Project Folder

```bash
cd student-management-system
```

---

###  Install Dependencies

```bash
pip install -r requirements.txt
```

---

###  Run Application

```bash
python app.py
```

---

###  Open Browser

```bash
http://127.0.0.1:5000
```
